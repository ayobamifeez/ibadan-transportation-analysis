# ibadan-transportation-analysis
Urban transportation analysis using Python, Excel, and Power BI
This repository contains all the materials used in my tutorial on YouTube.

## What's Inside
- **dataset/** – Raw and cleaned datasets used in the project
- **Pandas_code/** - The generated synthetic code
- **pbix/** – Final Power BI dashboard file  

## Requirements
- Excel or any tool that can open CSV files

## How to Use
1. Download the entire repository (top right → Code ).
2. Open the Excel file in Power BI.
3. Explore, modify, or recreate the visuals.
